# 模拟器管理

HiSH 支持创建多个模拟器实例，方便用户实现镜像制作、环境隔离等操作。同一时刻，用户可以选择任一实例运行。

在主界面下方选择“模拟器管理”即可进入“模拟器管理界面”，该界面展示了当前 HiSH 中的模拟器实例，并提供了 模拟器管理、镜像管理、共享文件夹管理
功能。

## 模拟器管理

模拟器管理界中，您可以点击标题栏右侧的“添加”按钮（PC则在列表下方），进入“编辑模拟器”界面，完成模拟器的基本配置后，即可点击“完成”完成模拟器创建。

在创建模拟器的界面中，可以设置：

* CPU核数：指定CPU核数
* 内存大小：指定内存大小
* 系统镜像：选择HiSH中已有的镜像创建模拟器
* 端口映射：将对主机端口的访问转发到模拟器端口中
* 只读共享文件夹
* init

在“模拟器管理”列表中，可以看到当前正在运行的模拟器实例，也可以其它点击卡片中的 “运行”
按钮启动其它模拟器，也可以直接将常用的模拟器实例设置为“默认启动”，即在HiSH启动时自动运行。

模拟器管理中还可以支持：

* 删除：删除模拟器实例及其对应的数据
* 复制：基于当前模拟器的配置和镜像复制一个新的模拟器实例
* 导出：保存将当前模拟器的文件系统镜像，用于后续创建新的模拟器时作为模板
* 修改：修改模拟器的运行设置，通常需要重启后才可生效
