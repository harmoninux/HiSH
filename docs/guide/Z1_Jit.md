# 运行加速

此版本HiSH默认使用“解释执行”的方式运行代码，运行效率受限。为实现更高的运行效率，HiSH提供了针对开发者的“开发版”，该版本中提供了JIT加速，相比解释执行，性能可提升5-6倍甚至更高。

开发版可以通过以下两种途径获取：

## 源码

从 开源仓库 获取源码，使用DevEco Studio编译、构建HiSH，并在开发者模式下安装到设备中。见：

- GitHub：https://github.com/harmoninux/HiSH
- GitCode：https://gitcode.com/realhackeris/HiSH
- Gitee：https://gitee.com/hackeris/HiSH

## hap包

从开源仓库的“Release页面”下载hap文件，在开发者模式下签名、安装。见：

- GitHub：https://github.com/harmoninux/HiSH/releases
- GitCode：https://gitcode.com/realhackeris/HiSH/releases
- Gitee：https://gitee.com/hackeris/HiSH/releases
